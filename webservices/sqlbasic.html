    Update Statement - עדכון

    
    UPDATE [Table]
    SET [Value] = 'Value'
    WHERE Condition...;

    מה שהקוד הזה בעצם עושה, זה מעדכן את הטבלה Table כך שבכל ערך שבו מתקיים Condition, הערך תחת השם Value ישתנה לערך 'Value'

    Insert Statement - הכנסה

    
    INSERT INTO [Table]
    ([Value1],[Value2],[Value3] ... )
    Values('Value1','Value2', ... );

    הקוד הזה יכניס ערך חדש לטבלה Table כך שלכל העמודות Value1, Value2 וכו... יכנסו ערכים מתאימים בValues דברים שיש לבדוק: לכל [Value] קיים ב Values ערך באותו מקום בדיוק צריך לוודא שלכל ערך יש '' מסביבו, ובין העריכים יש פסיקים
